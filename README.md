
# Event Management System

A web-based **Event Management System** developed using **PHP and MySQL**.  
This application allows users to create, manage, and view events with images through a simple and user-friendly interface.

---

## 📌 Features

- Create and manage events
- Upload event images
- Display events on the homepage
- PHP-based backend with MySQL database
- Simple and clean user interface
- Suitable for college mini projects

---

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** Apache (XAMPP / WAMP / LAMP)
